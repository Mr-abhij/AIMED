# AIMED - AI-based Medical Emergency Detection

A system for detecting medical emergencies using AI and mobile technology.

## Project Structure

```
aimed/
├── README.md           # This file
├── .github/           # GitHub workflows and actions
├── docs/              # Project documentation
├── infra/             # Infrastructure as Code (Docker, K8s, Terraform)
├── apps/              # Frontend applications
│   ├── mobile/        # Mobile app (Flutter/React Native)
│   ├── nurse_app/     # Nurse application (PWA/Mobile)
│   └── dashboard/     # Web dashboard (Next.js)
└── services/          # Backend services
    ├── backend/       # Main backend API
    ├── ml/            # Machine learning services
    └── workers/       # Background workers
```

## Getting Started

1. Clone the repository
2. Follow setup instructions in respective service directories
3. Run the applications
